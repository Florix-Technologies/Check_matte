# Checkmate Landing Page

A modern React landing page built with Vite, Tailwind CSS, and Framer Motion.

## Setup

1. Open the project folder in your terminal:
   ```bash
   cd f:/checkmate
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open the local URL displayed in your terminal.

## Project Structure

- `src/components` - shared UI components like `Navbar` and `Footer`
- `src/sections` - landing page sections like `Hero`, `Features`, `Testimonials`, and `CTA`
- `src/assets` - assets and illustrations

## Features

- React + Vite starter
- Tailwind CSS with `darkMode: 'class'`
- Dark/light theme toggle
- Glassmorphism and gradient visuals
- Responsive layout for mobile, tablet, desktop
- Smooth animations with Framer Motion
